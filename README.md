## File extensions
* .lib - бібліотека компонентів P-CAD
* .sch - схеми компонентів P-CAD
* .erc - розширення файлу звіту P-CAD

* .pcb - файл друкованих плат P-CAD
* .drc - файл перевірки друкованих плат P-CAD
